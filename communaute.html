Contribution

Comment contribuer à statAfrikR

📊 Utiliser statAfrikR

Installez le package, utilisez-le sur vos données réelles, partagez vos retours via GitHub Issues ou Discord. Chaque retour terrain oriente la roadmap.

install.packages("statAfrikR")
library(statAfrikR)

⚙️ Contribuer au code

Fork le repo GitHub, ouvrez une Issue, créez une branche de fonctionnalité, soumettez une Pull Request conforme au CONTRIBUTING.md. Tests unitaires obligatoires.

Voir CONTRIBUTING.md →

🎓 Devenir formateur accrédité

Obtenez la certification niveau 3 Expert, suivez le programme de formation de formateurs de la Foundation, et animez des sessions statAfrikR dans votre pays.
